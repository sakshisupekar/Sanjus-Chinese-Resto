# Sanju's Chinese Restaurant 🍜

A full-stack restaurant web application where admin 
can manage food menu and customers can register and place orders.

## Technologies Used
- Frontend: HTML, CSS, JavaScript, Angular
- Backend: Java, Spring Boot, REST APIs
- Database: MySQL

## Features
- Admin panel for food management
- User registration and login
- Menu display with prices
- Order placement system
- Responsive design

## How to Run
### Backend
1. Clone the repository
2. Open in Eclipse
3. Configure MySQL database
4. Run Spring Boot application

### Frontend
1. Navigate to frontend folder
2. Run `npm install`
3. Run `ng serve`
